# ladder-v2 扩展难度阶梯

`ladder-v2` 保留 `ladder-v1` 的五道基准题，并为每个等级增加两道不同主考点的题，总计 15 题。所有题权重均为 `1.0`，最大背包为 5×5，并且都有 60 秒预算内完成的精确 Oracle。

## 能力覆盖矩阵

| 等级 | 题目 | 唯一主考点 | Oracle |
|---|---|---|---:|
| L1 | 宝石夹击 | 对称的左右相邻关系 | 16 |
| L1 | 九格拼刃 | 无效果干扰的精确填满与多形状旋转 | 17 |
| L1 | 转角罗盘 | 非对称 L 形方向随物品旋转 | 16 |
| L2 | 缺角背包的取舍 | 不规则背包、超容量与弃物选择 | 32 |
| L2 | 四向共振 | 同一来源对同一目标的多接触重复结算 | 25 |
| L2 | 双核聚焦 | 两个独立来源的正交影响区与叠加 | 28 |
| L3 | 交叉火力 | 局部相邻与可穿透射线联合规划 | 50 |
| L3 | 双路遮光 | 正面射线遇到第一个物品停止，考察前后顺序 | 27 |
| L3 | 平行穿透 | 多格来源的平行射线、穿透与重复命中 | 35 |
| L4 | 诅咒与阻挡 | 负面邻接、正面阻挡射线和主动留空 | 52 |
| L4 | 诅咒屏风 | 固定锚点产生负面射线，用无属性屏风吸收 | 37 |
| L4 | 日蚀交叉场 | 正负两套可穿透射线的来源分离 | 74 |
| L5 | 棱镜迷阵 | 同一物品正负双效果、旋转诅咒与挡板诱饵 | 71 |
| L5 | 星位偏移网 | 两格远稀疏偏移、对角惩罚和多效果耦合 | 71 |
| L5 | 光束迷宫 | 重复穿透正光束、可阻挡负光束和超容量选择 | 92 |

“唯一主考点”表示题目用来诊断的主要能力不同；高等级仍会自然复用低等级的坐标、旋转和合法拼装能力。结构测试会检查每级恰好三题、主考点标签不重复、所有 Oracle 可回放以及背包不超过 5×5。

## 使用

```powershell
uv run bbbench suite validate .\suites\ladder-v2.yaml
uv run bbbench run .\configs\run.ladder-v2.yaml --dry-run
uv run bbbench web
```

示例模型配置下，批跑规模为 `2 模型 × 15 题 × 3 trials = 90 jobs`。Web 实验台会自动发现新 suite 和 run 配置。
